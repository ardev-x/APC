# Arbitrary Precision Calculator (APC)

APC is an Arbitrary Precision Calculator implemented in C, designed to perform arithmetic operations on large integers with precision beyond native data type constraints.

## Project Status

🚧 This project is currently under development.

## Features

- Perform arithmetic operations on arbitrarily large integers
- Modular project structure
- Implemented in C
- Uses linked lists for large number representation
